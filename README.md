# 三国乱世 - AI 驱动的角色扮演游戏

基于 LangGraph 和 RAG 的三国题材文字 RPG 系统

## 🎮 游戏简介

这是一款 AI 驱动的三国题材文字角色扮演游戏。你将穿越到东汉末年，在黄巾起义、群雄割据的乱世中，选择自己的道路，书写属于你的传奇。

**特色功能：**
- 🤖 **AI 叙事者**：动态生成符合历史背景的剧情和事件
- 📚 **RAG 检索增强**：基于真实三国历史，确保剧情合理性
- 🎭 **角色扮演**：选择不同身份（武将、谋士、君主等），体验不同的人生轨迹
- ⚔️ **动态事件**：你的每个决策都会影响历史走向
- 🎯 **参考策略**：AI 根据你的角色性格提供符合人设的行动建议

## 📦 快速开始

### 1. 安装依赖

```powershell
pip install -r requirements.txt
```

**注意**：安装完成后 IDE 的"未解析的引用"错误会自动消失。

### 2. 配置环境变量

编辑 `.env` 文件，设置你的智谱 API 密钥：

```bash
ZHIPU_API_KEY=your_actual_api_key_here
```

### 3. 启动应用

```powershell
uvicorn app.main:app --reload
```

访问 http://127.0.0.1:8000 查看 API 文档。

## 🎯 核心功能

- **FastAPI** 应用框架
- **LangGraph** 状态机（4 个核心 Agent）：
  - 🎭 **主持人**：推进剧情，生成动态事件，提供互动选项
  - 🔍 **侦探**：分析局势，提供策略建议
  - 👁️ **证人（NPC）**：提供情报、任务、线索的 NPC 互动
  - 💭 **角色内心**：表达角色志向和内心独白
- **RAG** 检索增强生成（基于真实三国历史）
- **智谱 AI** 大模型集成
- **动态事件系统**：根据时间、地点和玩家行为实时生成事件
- **角色驱动系统**：不同身份有不同的语言风格和行为模式

## 🎮 游戏玩法

### 角色创建
游戏开始时，你可以从 RAG 系统生成的三国历史人物中选择角色：
- **武将型**：关羽、张飞、赵云等（擅长征战）
- **谋士型**：诸葛亮、郭嘉、周瑜等（擅长策略）
- **君主型**：刘备、曹操、孙权等（擅长统御）
- **其他类型**：商人、官员、义士等

### 游戏流程
1. **剧情推进**：AI 叙事者描述当前局势和发生的事件
2. **互动选项**：系统提供 2-3 个行动选项（格式统一为"你可以..."）
3. **参考策略**：根据你的角色性格，生成符合人设的策略建议
4. **玩家决策**：选择参考策略或自定义行动
5. **事件结算**：根据你的决策，生成新的剧情和游戏效果

### 示例对话

```
【剧情推进】
孙坚站在宛城附近的山岗上，眺望着远方的曹军先锋。他的眉头紧锁，
心中盘算着如何应对眼前的局势。

【互动选项】
选项一 派遣斥候侦查宛城的防御情况...
选项二 亲自带领精锐部队夜袭曹军先锋...
选项三 与部下商议制定更周全的计划...

📜 参考策略 (符合孙坚的性格和立场):
------------------------------------------------------------
  [1] 末将愿领兵前往，探查敌情！
  [2] 某家不才，愿为先锋，直取敌将首级！
  [3] 此事非同小可，还望从长计议。
------------------------------------------------------------

请输入您的行动:
  - 直接输入文字使用自定义行动
```

## 🎲 游戏系统详解

### 1. 动态事件系统
- 基于时间、地点自动生成事件
- 考虑玩家行为和当前局势
- 事件类型：战场、政治、外交、民生等

### 2. RAG 检索系统
- 从三国历史中检索相关事件和人物
- 确保剧情的历史合理性
- 支持父子文档检索（更精准的上下文）

### 3. 角色驱动系统
- 不同身份有不同的语言风格
  - 武将："末将愿"、"某家当"
  - 谋士："在下建议"、"臣以为"
  - 君主："孤可"、"朕当"
- 决策会影响角色声望和势力

### 4. 参考策略对齐
- 自动从主持人选项中提取内容
- 转换为符合角色口吻的策略
- 支持多种选项格式（正则解析）
